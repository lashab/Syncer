# Syncer

## INTRODUCTION

Module allows to export and import content in YAML format.

## INSTALLATION

<code>composer require drupal/syncer</code>

## REQUIREMENTS
ZipArchive extension

## CONFIGURATION
The module has no modifiable settings.
